# imagetools

A simple Python image processing toolkit built using **Pillow + OpenCV**.

It provides basic image editing, restoration, and enhancement features in a clean and easy-to-use API.


##  Features

### Basic Image Operations (Pillow)
- Grayscale conversion
- Resize images
- Rotate images
- Blur effect
- Brightness adjustment
- Edge detection

### Advanced Tools (OpenCV)
- Object removal (mask-based inpainting)
- Image restoration (denoise + sharpen + contrast)
- Image cleanup pipeline
- Image quality scoring
